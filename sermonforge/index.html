SermonForge

A sermon preparation workspace for pastors. Built for the full arc of study — exegesis, structure, manuscript, and delivery.

Download for Windows Free · Windows 10 or later · 64-bit
First Run Windows will show a blue "Windows protected your PC" warning because the installer isn't yet signed. Click More info, then Run anyway. This is normal for new apps and doesn't mean anything is wrong.
Requires A Claude API key from console.anthropic.com. Enter it on first launch — no account or subscription beyond that.
Updates Automatic. The app checks for new releases on launch and installs them silently in the background.
